# Scripts

This folder is meant to hold some useful scripts related to the main project, such as the script we use to generate a table of bitboards that represent possible moves for a queen.

These scripts are written in Go, and naturally require the [Go compiler](https://go.dev/) to build. If you have it installed, you can move into this directory and then run 

```sh
go build .
```

to compile the scripts into an executable. To use the scripts, just run the new executable. It will include help commands to guide you through the usage.
